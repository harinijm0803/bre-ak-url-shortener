Bre.ak – URL Shortener Web Application

Bre.ak is a simple and clean web-based URL shortener built using Django. 
This system allows users to generate short links for long URLs and manage them through a dashboard interface. 
Users can create new short links, view all generated links, and quickly open the original website using the shortened URL.
The dashboard displays the website icon, original URL, and the shortened link in an organized layout. Each shortened link is unique and redirects directly to the original website when clicked. The system is designed with a minimal UI, light color theme, and an easy-to-use interface.This project was developed as a web application using Python and Django with SQLite database.

Features

Dashboard Interface: View all created links in a clean dashboard layout
URL Shortening: Convert long URLs into short links like bre.ak/XXXXX
Auto Redirect: Clicking the short link opens the original URL
Website Icon Display: Shows favicon of each URL in dashboard
Create New URL Page: Generate new short link using original URL
Delete Links: Remove unwanted links from dashboard
Unique Short Code: Automatically generated short codes
Simple UI Design: Light color theme with centered input box
SQLite Database: Stores URLs and short codes
Django Backend: Built using Django framework


Tech Stack

Backend: Python, Django  
Frontend: HTML, CSS  
Database: SQLite  
Version Control: Git, GitHub  


Usage

1. Open dashboard
2. Click "Add URL" in the top right corner
3. Paste original link
4. Click Generate
5. Short link will appear in dashboard
6. Click bre.ak/XXXXX to open original site
