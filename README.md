# **Bre.ak – URL Shortener Web Application**

Bre.ak is a simple and clean web-based URL shortener built using Django.  
This system allows users to generate short links for long URLs and manage them through a dashboard interface.  
Users can create new short links, view all generated links, and open the original website using the shortened URL.
The dashboard displays the website icon, original URL, and the shortened link in an organized layout.  
Each shortened link is unique and redirects directly to the original website when clicked.  
The system is designed with a minimal UI, light color theme, and an easy-to-use interface.

## Features

- Dashboard Interface to view all links  
- URL Shortening (bre.ak/XXXXX)  
- Auto redirect to original URL  
- Website icon shown for each link  
- Create new URL page  
- Delete links from dashboard  
- Unique short code generation  
- Simple light UI design  
- SQLite database storage  
- Django backend

## Tech Stack

- Python  
- Django  
- HTML  
- CSS  
- SQLite  
- Git  
- GitHub  

## Usage

1. Open the dashboard  
2. Click **Add URL** in the top right corner  
3. Paste the original link  
4. Click Generate  
5. Short link will appear in dashboard  
6. Click bre.ak/XXXXX to open the original website
